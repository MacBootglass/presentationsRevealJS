Fonctionnalités disponibles

Module navbar

Barre de navigation resonsive et animée référençant l’ensembles des modules dont le référencement aura été préalablement autorisé.

Exemple de rendu sur Mobile

Module home

Ecran d'accueil composé d'un fond animé, d’une photo centrale et une zone de texte animée.

Exemple de rendu sur Desktop

Module about

Se compose d'un titre (nom?), d'une zone de texte (description?), d'une liste de liens sous la forme d'icones, et d'un bouton permettant le téléchargement d'un fichier (CV?).

Exemple de rendu sur Desktop

Module project_list

Permet de lister des projets. Chacun d'entre eux dispose d'un titre, d'une courte description ainsi qu’un bouton (optionnel) de redirection.

Exemple de rendu sur Desktop
Exemple de rendu sur Mobile

Module skill_list

Se décompose en catégories de compétences. Chacune d'entre elles est composé d'une liste de compétences (nom + niveau [0;100]). Les barres de progressions sont animées

Exemple de rendu sur Desktop
Exemple de rendu sur Mobile

Module rubrique

Permet de lister des articles décomposés en blocs. Chacun d'entre eux comprant un titre ainsi qu’un contenu textuel au format markdown. Une timeline est générée sur le côté de ces articles. Une date peut ainsi être indiquée à côté des articles.

Exemple de rendu sur Desktop
Exemple de rendu sur Mobile

Module footer

Permet de disposer d'un pied de page décomposé en deux zones de texte et comprenant un bouton permettant de remonter en haut de page.

Exemple de rendu sur Desktop

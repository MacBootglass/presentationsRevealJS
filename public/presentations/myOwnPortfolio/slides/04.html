Module navbar

Barre de navigation resonsive référençant l’ensembles des modules dont le référencement aura été préalablement autorisé.

Si résolution <= 900px, des icônes (librairie font-awesome) référencés dans un fichier de configuration remplacent l'affichage standard. Lors d’un scroll vers le bas de l’écran, la barre de navigation disparait automatiquement, et réapparait lors d’un scroll vers le haut.
